# pine v0.4.0

## What's New

* **Instant library loading** — the library view now shares the same local cache as the command palette, so switching libraries or opening a library feels immediate.
* **`--clear-cache` flag** — run `pine --clear-cache` to wipe the local API/search cache and exit. Useful when cache data looks stale or after server-side changes.
* **Delete items from the server** — remove books or podcast episodes directly from the detail view with a confirmation overlay. Deleted items are evicted from the local cache immediately.
* **Background cache pre-warm** — pine keeps the cache warm in the background, reducing wait time when you first open a library.

## Fixes

* Resolved cache race conditions and missing cache prebuilds on library open.
* Cache-ready messages are now routed to the active screen correctly.
* Legacy/empty SQLite schemas are ignored instead of crashing on startup.
* Fixed a search cache index limit collision that could return wrong items.
* Deleted episodes and items are removed from home/library caches immediately.
* Corrected the podcast episode delete flow and help bar text clipping.
* Prevented `esc` from exiting views unexpectedly.
* Worked around a Ghostty `AltGr` typing issue for international characters.
* Truncated long titles in the player footer to prevent line wrapping.
* Fixed text offset shifting in overlays.

## Installation

```sh
go install github.com/Thelost77/pine@v0.4.0
```

## Requirements

- Go 1.25+
- [mpv](https://mpv.io/) installed and available in `PATH`
- An [Audiobookshelf](https://www.audiobookshelf.org/) server
