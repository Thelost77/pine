# pine v0.3.0

## What's New

* **Persistent API Cache**: Experience instant startup and near-instant library switching. Data is cached locally and updated intelligently.
* **Metadata Editor**: Edit book and episode metadata directly within pine.
* **Search Palette**: The dedicated search screen has been replaced by a sleek, ubiquitous search palette accessible from anywhere.

## Fixes

* **Terminal Scroll Bug**: Fixed a severe rendering glitch that caused the UI to duplicate and interleave (the "doubled" view) when switching libraries rapidly. This was caused by terminal auto-wrapping on the bottom-right corner in macOS Terminal and iTerm2.
* **Podcast Deduplication**: Fixed an issue where podcast episodes in "Continue Listening" were not properly hidden from the "Recently Added" shelf.
* **MPV Socket Path**: Fixed an issue with double slashes in the mpv IPC socket path.
* **Search Palette Routing**: Fixed a bug where searching for a series incorrectly routed to the detail view instead of the series view.
