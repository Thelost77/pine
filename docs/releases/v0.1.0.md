## pine v0.1.0

First public release of `pine`, a terminal UI client for Audiobookshelf.

### Highlights

- Browse audiobook and podcast libraries from the terminal
- Playback with chapter navigation and multi-track support
- Search across your libraries
- Bookmarks, sleep timer, and session restore
- Automatic progress sync back to Audiobookshelf

### Installation

```sh
go install github.com/Thelost77/pine@v0.1.0
```

### Requirements

- Go 1.25+
- `mpv` available in `PATH`
- An Audiobookshelf server

### Notes

- Config, local data, and logs live under `~/.config/pine`
- This is the initial public release, so some rough edges are expected
