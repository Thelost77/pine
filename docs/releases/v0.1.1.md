## pine v0.1.1

Patch release focused on making session restore safer for stale playback
state.

### Highlights

- Skip startup restore silently when a saved item or podcast episode no
  longer exists
- Clear stale saved sessions after restore failures so broken state does
  not repeat on every launch
- Avoid falling back to item-level podcast playback when the saved episode
  cannot be resolved

### Installation

```sh
go install github.com/Thelost77/pine@v0.1.1
```

### Requirements

- Go 1.25+
- `mpv` available in `PATH`
- An Audiobookshelf server

### Notes

- Existing installs can update in place with `go install ...@v0.1.1`
- This release fixes restore behavior when media was removed after a
  session had already been saved
