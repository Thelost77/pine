## pine v0.2.0

Feature release adding a command palette for quick navigation, search, and
actions.

### Highlights

- **Command palette** (`Ctrl+P`) — a "Search Everywhere" modal with grouped
  static actions (navigation, playback, sleep timer), screen-contextual
  actions, and fuzzy content search against the local cache
- `j/k`, arrow keys, and `Ctrl+J/K` navigate the palette list; typing any
  character auto-focuses the search input
- Series data is now indexed in the search cache alongside library items,
  making series discoverable from the palette
- Pressing arrow keys or `Ctrl+J/K` from the search input defocuses back
  to the result list

### Other changes

- Search cache extended to fetch and index series metadata
- Screen models expose `SelectedPaletteActions()` for contextual items
- Fixed palette dialog height stability and item text truncation

### Installation

```sh
go install github.com/Thelost77/pine@v0.2.0
```

### Requirements

- Go 1.25+
- `mpv` available in `PATH`
- An Audiobookshelf server

### Notes

- Existing installs can update in place with `go install ...@v0.2.0`
